# Top 500 Interview Questions

## Overview
This document consolidates a broad set of questions often used during Java Full Stack interviews.

## Question Categories
1. Java fundamentals and OOP
2. Collections and concurrency
3. JVM, GC, and performance
4. Spring, Hibernate, and JPA
5. REST APIs and security
6. SQL and database design
7. Messaging and caching
8. Frontend and TypeScript
9. System design and cloud
10. Git and debugging

## How to Use This List
- Start with fundamentals before moving into architecture topics.
- Group questions by area and practice them in sequence.
- Review the reasoning behind answers, not just keywords.
